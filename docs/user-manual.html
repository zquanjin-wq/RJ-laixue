一、入门

1.1 访问入口

打开 https://www.laixue.work，输入访问码 9B7719 进入登录页。第一次使用时，需要管理员提前给你创建账号并告知初始密码。

1.2 三类账号

角色谁能做什么
admin（管理员）管账号（创建学员/老师）+ 管课件 + 编辑/预览所有课件
teacher（课件创作者）创建/编辑自己的课件，分享给学员
learner（学员）查看分配给自己（或全部公开）的课件，参与互动学习

1.3 登录后看到什么

角色登录后进入
admin管理后台（学员/老师/课件管理）+ 创作首页
teacher创作首页（生成课件、保存到云端、编辑已发布课件）
learner学员首页（看到所有可学习的课件 + 进入教室）

二、课件创作者（admin / teacher）

2.1 创建一门新课

1登录 → 创作首页。在输入框里写课程主题，例如"新员工入职培训"或"结构化思考：OBER 法则"。
2点击生成。AI 会自动产出课程大纲 → 逐页内容 → 配图 → TTS 语音讲解。整个过程大约 3-10 分钟。
3预览。生成完毕后可在首页直接预览课件播放效果。
4保存到云端。点击"保存到云端"按钮，课件进入你的"我的课程"列表。
5分享。在"我的课程"里点击"分享"按钮 → 链接已复制 → 发给学员或粘贴到飞书 / 邮件。

2.2 编辑已发布的课件

1在首页"我的课程"里，悬停课程卡片 → 右上角出现编辑按钮（铅笔图标）。
2点击铅笔 → 进入专业模式（Pro Mode）。可以修改每页文字、配图、AI 讲解内容。
3改完后点"保存到云端" → 同步给所有学员。

2.3 删除课件

只有课件创建者能删除。在"我的课程"里点删除按钮 → 确认 → 即刻从云端移除。

⚠️ 删除不可撤销。学员已经收到的分享链接会失效。

三、运营管理员（admin）

3.1 学员管理

1进入"管理后台" → 学员管理。
2顶部"创建新学员"表单 → 输入姓名 + 邮箱 → 提交。系统会生成一个 12 位初始密码，绿色卡片显示出来，请立刻抄下告诉学员。
3列表里可对每个学员操作：重置密码 / 禁用 / 启用 / 删除。禁用后学员登录会看到"账号已停用"卡片。

3.2 老师管理

与学员管理类似。老师账号可以创建/编辑自己的课件，但不能管其他账号。

3.3 课件管理

查看所有云端已发布的课件，点击"进入"以只读模式预览。

3.4 常见操作清单

场景操作
新员工入职创建学员账号 → 抄下初始密码 → 口头/飞书发给员工
老师需要备课权限创建老师账号 → 老师登录后即可在首页生成课件
员工离职学员列表 → 禁用（或删除）该员工
员工忘记密码学员列表 → 重置密码 → 把新密码发给员工

四、学员

4.1 开始学习

1收到分享链接（例如来自 admin 或老师发的飞书消息） → 点击打开。
2如果尚未登录，会跳转到登录页 → 用 admin 提供的邮箱 + 初始密码登录 → 自动跳回课程。
3课件开始播放，AI 讲师讲解内容。

4.2 互动方式

操作效果
点击"下一步"按钮翻到下一页
在右下方输入问题AI 老师立即回答你的问题
观看 AI 老师讲解语音 + 文字同步显示，可以重听

4.3 常见问题（学员视角）

Q: 我点链接后要求输入访问码？
A: 这是早期的访问码（9B7719）验证。如果你的账号已创建，会跳转到登录页；账号验证通过后直接进入课程。

Q: 我的账号被禁用了？
A: 联系管理员（通常是培训部门同事），让他们在学员管理里重新启用你的账号。

Q: 我能跳过某页吗？
A: 可以。点击"下一步"按钮直接翻页。

五、常见问题

5.1 我分享了课程链接，对方说打不开？

最常见原因：对方没有登录。让他们用 admin 提供的账号先登录（不需要退出当前页面），链接会自动重新跳转。

5.2 生成的课件质量不理想？

在专业模式（点击铅笔）里手动修改每页的文字、配图、AI 讲解。AI 生成的是初稿，人工精修是必走环节。

5.3 我能让学员只看到某些课程，不看到全部吗？

当前阶段所有 active 学员可以看到所有云端课件。后续会支持按学员分配。

5.4 课程生成要等多久？

取决于页数。一般 5 页课程约 3-5 分钟，10 页课程约 8-12 分钟。生成过程中可以最小化窗口，不影响后续使用。

5.5 数据安全吗？

所有课程数据和学员学习记录存在 Supabase PostgreSQL，通过行级安全（RLS）严格隔离。每个学员只能看到自己的数据，admin 可以看到全部。详见技术文档。
